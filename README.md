# TT-IC-Editing Android图片编辑应用

## 项目简介
基于Android开发的轻量级图片编辑工具，支持裁剪、文字添加、滤镜等功能。

## 技术栈
- 开发语言：Java
- 开发环境：Android Studio
- 最低支持：Android 10 (API 29)
- 核心库：Glide、FileProvider

## 功能特性
- ✅ 相册选取图片
- ✅ 相机拍照
- ✅ 图片裁剪（1:1、4:3、16:9等比例）
- ✅ 旋转与翻转
- ✅ 亮度与对比度调节
- ✅ 文字编辑模块
- ✅ 贴纸功能

## 安装运行
1. Clone项目到本地
2. 使用Android Studio打开
3. 连接Android设备或启动模拟器
4. 点击Run运行项目
